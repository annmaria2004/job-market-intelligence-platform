
# Global AI & Data Science Job Market Intelligence Platform

## Overview
A full-stack analytics platform that helps users explore AI and Data Science job trends worldwide.

## Features
- 50,000 AI/Data Science job records
- MongoDB Atlas cloud database
- FastAPI backend
- Salary analytics
- Country-wise job distribution
- Remote/Hybrid/Onsite analysis
- Experience-level distribution

## Tech Stack
- Python
- FastAPI
- MongoDB Atlas
- React (Upcoming)
- Render (Upcoming)
- Vercel (Upcoming)
